# code-canvas

序言-xy 个人主页 — 技术展示、项目集、产品引流

## 项目目标

- 展示个人技术栈、项目和产品
- SEO 优化，提升搜索引擎可见度
- 引流至公众号及其他产品，构建流量闭环
- Canvas 技术学习与实验

## Tech Stack

- Next.js 16 (App Router, Turbopack)
- React 19 + TypeScript
- Tailwind CSS v4
- daisyUI 5

## 开发

```bash
pnpm dev          # http://localhost:3000
pnpm build        # 生产构建
pnpm lint         # ESLint
```

## 页面结构

| 路由 | 用途 |
|------|------|
| `/` | 主页 — 个人介绍、技术栈、项目展示、引流入口 |
| `/canvas` | Canvas 学习实验页 |
